# React Project Skill

Reference skill for a web React project (Vite/Next). The Verifier uses these gates.

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Tests

```bash
npm test
```

## Conventions

- Functional components + hooks only.
- Colocate styles with components; prefer CSS Modules / Tailwind over global CSS.
- Keep presentational and data-fetching logic separated.
